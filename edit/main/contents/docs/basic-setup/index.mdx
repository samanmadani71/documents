## مستندات

**مستندات** یک کیت آغازین سبک و ماژولار است که با **Next.js**، **React**، **Tailwind CSS** و **TypeScript** ساخته شده است. هدف آن این است که بدون صرف زمان برای کدهای تکراری، مستندات حرفه‌ای و ساخت‌یافته را سریعاً راه‌اندازی کنید.

فرقی نمی‌کند راهنمای محصول می‌نویسید، اسناد سیستم‌های داخلی را تدوین می‌کنید یا مستندات توسعه می‌سازید؛ مستندات یک پایه تمیز، قابل‌گسترش و نگهداری‌پذیر در اختیارتان می‌گذارد.

## چرا از مستندات استفاده کنیم؟

این کیت برای ساده‌سازی تولید و انتشار مستندات ساخته شده است. از پروژه‌های کوچک (یک README) تا پایگاه‌های دانش سازمانی مقیاس‌پذیر است.

با MDX، کامپوننت‌های قابل‌استفادهٔ مجدد و استایل‌دهی Tailwind، جریان کاری شما را کارآمد نگه می‌دارد بدون اینکه از انعطاف‌پذیری کم کند.

از آن استفاده کنید برای:

- **راهنماهای محصول** – راه‌اندازی، استفاده و رفع اشکال
- **مستندات داخلی** – فرآیندها، سیاست‌ها و مراجع تیمی
- **راهنماهای فنی** – APIها، SDKها و مرور معماری

رابط کاربری و ساختار را مطابق برند یا فرایند کاری خود سفارشی کنید—هیچ قفل فناوری‌ای وجود ندارد.

## قابلیت‌های اصلی

| قابلیت                     | توضیح                                                             |
| -------------------------- | ----------------------------------------------------------------- |
| **نوشتن با Markdown/MDX**  | ترکیب Markdown با کامپوننت‌ها و Mermaid.js برای محتوای غنی.       |
| **ناوبری انعطاف‌پذیر**     | منوهای چندسطحی، سلسله‌مراتب صفحات و فهرست خودکار مطالب.           |
| **پشتیبانی از کد**         | برجسته‌سازی نحوی، تب‌های کد و کپی یک‌کلیکی.                       |
| **جست‌وجو**                | تطبیق فازی با هایلایت و نتایج آنی.                                |
| **طراحی واکنش‌گرا**        | حالت روشن/تاریک داخلی و پشتیبانی کامل موبایل.                     |
| **ریاضی و جدول‌ها**        | رندر LaTeX و استایل تمیز برای جدول‌ها.                            |
| **پیش‌فرض‌های SEO**        | فراداده از پیش‌پیکربندی‌شده، Open Graph و داده‌های ساخت‌یافته.    |
| **اسناد هوشمند (به‌زودی)** | ابزارهای هوش مصنوعی برای جست‌وجوی هوشمند و تولید محتوا (به‌زودی). |

## استفاده از این کیت

ناوبری در سمت چپ قرار دارد. صفحات از راه‌اندازی تا قابلیت‌های پیشرفته پیش می‌روند، اما می‌توانید آزادانه بین بخش‌ها جابه‌جا شوید.

از فهرست مطالب سمت راست برای پرش به بخش‌های مختلف استفاده کنید.

با [راهنمای نصب](/docs/basic-setup/installation) شروع کنید.
